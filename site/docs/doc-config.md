#Options

You can customize Simditor by passing optional options when initializing Simditor.
Here are some optional options and their default values:

```coffee
editor = new Simditor
  textarea: null
  placeholder: ''
  defaultImage: 'images/image.png'
  params: {}
  upload: false
  tabIndent: true
  toolbar: true
  toolbarFloat: true
  toolbarFloatOffset: 0
  toolbarHidden: false
  pasteImage: false
  cleanPaste: false
  autolink: false
```
